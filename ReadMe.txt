#Read Me
The .PDF file is the report of this practice, while the folder is the project. 
To launch the game, you could open the phidget.pde with Processing, or launch with the .exe file in the application folder ( only if your system is Windows)
To play the game, you should connect your phidget kit to your computer, including following components:
1.Mini stick sensor(voltage input of channel 0 and digital input of channel 0 as digital input) ;
2.Rotation sensor(voltage input of channel 4) ;
3.Indoor light sensor(voltage input of channel 6) ;
4.LEDs(Red one on digital output of channel 0, green one of channel 1, yellow one of channel 2).
